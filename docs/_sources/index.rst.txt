YouTube Trend Analysis Documentation
=====================================

Welcome to the YouTube Trend Analysis Documentation!🙌

This document is created as part of the final project for the 4th batch of the Programmers Data Engineering Dev Course.


📌 Project Overview
--------------------

**'YouTube Trend Analysis'** is a project aimed at analyzing trending YouTube videos based on various factors such as views, likes, comments, and sentiment analysis.
The goal is to predict future trends in the YouTube platform and provide insights into what types of content are gaining popularity.

Project Duration: From December 5, 2024, to January 15, 2025


Key Elements
^^^^^^^^^^^^^
- `YouTube API Usage` :
   Collect and analyze data such as popular videos, comments, and categories from YouTube.
- `Data Pipeline` :
   Utilize Apache Airflow, Spark, and DBT to collect and transform data.
- `ML Models` :
   Perform sentiment analysis on comments using Natural Language Processing (NLP) techniques.
- `Dashboard` :
   Visualize the analyzed data using Tableau to track trends in real-time.
